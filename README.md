# Potree Annotation Manager

This project is a **3D point cloud viewer** using **Potree** with annotation support.  
Annotations are managed through a **Node.js backend API** with CRUD operations.

---

## Features

- View 3D point clouds using Potree.
- Add annotations by clicking on points.
- Fly to annotations by selecting from a list.
- Delete individual annotations or clear all.
- Backend API stores annotations in MongoDB.
